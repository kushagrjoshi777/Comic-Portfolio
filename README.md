# Comic Portfolio — First Draft

An immersive, interactive comic-book style portfolio experience designed as a multi-page spread on a physical desk.

## Project Vision
This project transforms a standard professional portfolio into a tactile, expressive comic book spread. Users navigate through jagged panels and vibrant spreads that react to interaction with high-impact visual effects.

## Initial Implementation Plan
The first draft focuses on the core visual language and interaction engine:
- **Interactive Panels**: Implementing a dynamic grayscale-to-color transition when panels are hovered over (JoJo-inspired palettes).
- **Visual Polish**: Refining the "Realistic Desk" aesthetic, including lamp spotlighting, beige grainy paper textures, and physical depth (gutter shadows).
- **Component Imagery**: Adding high-quality, stylized images to different components to enhance the storytelling aspect of each section.
- **Impact Animations**: Using GSAP to trigger "POW!", "SKADOOSHH!", and "WAKAAUU!" effects on panel interaction.
